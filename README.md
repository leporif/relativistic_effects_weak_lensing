# relativistic_effects_weak_lensing
This repository contains the codes used to compute theoretical predictions for weak-lensing angular power spectra at linear and second order in cosmological perturbation theory, including the contribution from frame-dragging effects.

The codes reproduce the results presented in the paper:

“Perturbative and numerical study of nonlinear relativistic effects in weak lensing”
by M. Magi, F. Lepori, J. Adamek.

The repository provides both the analytical predictions and the simulation measurements used to generate the figures in the paper.

## Repository Structure

The repository is organized as follows:

```
.
├── sherBmodes-theory.ipynb
├── Figures.ipynb
├── sim-data/
├── theory/
├── Figures/
└── environment.yml
```

**sherBmodes-theory.ipynb**  
Jupyter notebook that computes the scalar-induced second-order contributions to the shear B-modes and ellipticity.  
These calculations involve multidimensional integrals and can be computationally time-consuming.

**Figures.ipynb**  
Jupyter notebook that reproduces the figures presented in the paper.  
It also computes, on the fly, several theoretical predictions that are less computationally expensive, including the frame-dragging power spectra.

**sim-data/**  
Directory containing the angular power spectra measured from simulations.  
These spectra are extracted from simulated maps generated using the ray-tracing tool available at:  
https://github.com/JulianAdamek/snapshot-raytracer

**theory/**  
Directory containing precomputed theoretical predictions that are loaded by `Figures.ipynb`.

**Figures/**  
Directory where the figures generated by the notebooks are stored.

**environment.yml**  
Conda environment file specifying the Python packages and versions required to run the notebooks.

## Acknowledgements

Parts of the theoretical calculations are adapted from the CAMB code
(Code for Anisotropies in the Microwave Background) developed by
Antony Lewis and Anthony Challinor (https://camb.info). 

The implementation follows the methodology described in
arXiv:1605.05662.

Some theoretical predictions are computed using the CLASS cosmology code (http://class-code.net/).

The simulated weak-lensing maps were generated with the ray-tracing
tool available at:
https://github.com/JulianAdamek/snapshot-raytracer
